# SCUCACompQQbot

## Introduction - 介绍

本项目为四川大学魔方协会SCUCAComp（周赛网站）的QQ机器人，提供协会周赛相关信息查询，选手成绩比较等功能。更多关于协会的信息，
请访问[协会网站](https://scu.yougi.top/)。

## Features - 功能

- 提供周赛相关信息查询
- 提供其他相关功能
